cargarPeliculas()

funcion "cargar peliculas" Hacec una peticion a la API de películas a través de un fetch. Almacenamos los resultados JSON en una variable para seguidamente hacer un foreach y poder recorrerlos uno a uno junto con un contador para adjudicarles a cada uno un div. Paralelamente en la API el endpoint o fetch tiene la opcion de la paginación.
